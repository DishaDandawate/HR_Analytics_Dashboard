# 📊 HR Analytics Dashboard

## 📌 Objective
To analyze employee data and identify key factors affecting attrition and performance.

## 🧠 Business Problem
High employee attrition impacts company performance. This dashboard helps HR understand patterns and take better decisions.

## 🛠 Tools Used
- Power BI
- Excel

## 📊 Dashboard Features
- Employee attrition analysis
- Department-wise performance
- Salary and experience insights
- Interactive filters for better analysis

## 📈 Key Insights
- Higher attrition observed in certain departments
- Employees with lower salary show higher turnover
- Experience level impacts retention
- Some roles have higher churn rate
